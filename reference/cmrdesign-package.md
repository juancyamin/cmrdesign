# cmrdesign: Conditional Minimax Regret design rules

Applied Conditional Minimax Regret (CMR) design rules for pilot-informed
experiments. The package includes two-arm, shared-control multi-arm,
stratified, multiple-outcome, proxy/delayed-outcome, unbounded-outcome,
and pilot-planning workflows.
